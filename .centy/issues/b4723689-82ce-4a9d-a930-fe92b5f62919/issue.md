# Create SSG Next.js landing page webapp

Build a Static Site Generation (SSG) Next.js webapp for the Centy landing page.

## Requirements
- Next.js with SSG (Static Site Generation) for optimal performance
- Deploy target: Cloudflare Pages
- Purpose: Landing page for the Centy project

## Technical Notes
- Use next export or App Router with static generation
- Configure for Cloudflare Pages deployment
- Ensure compatibility with Cloudflare's edge network
